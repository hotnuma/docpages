  • Description

    An action runs typically from Clipboard, DnD or Launcher, which will call an Application function which will run an io_job function in a ThunarJob thread.

    For example, the _launcher_action_delete will call application_unlink_files which calls the following :

    _launcher_action_delete
    application_unlink_files
        _application_launch
            io_unlink_files
                simple_job
                    _io_unlink
                        _io_delete_file
                            g_file_delete

VnrWindow

  • Description

    VnrWindow is a GtkWindow. It contains a UniScrollView which contains an UniAnimView to display the current image.

    UniAnimView is a UniImageView widget, it can display a GdkPixbufAnimation or a static GdkPixbuf.

    UniImageView contains a UniDragger to move the current image, UniDragger contains a UniPixbufDrawCache.
